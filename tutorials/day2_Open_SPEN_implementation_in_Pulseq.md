# Open-SPEN Implementation in Pulseq

The **Open-SPEN** implementation provides tools and scripts for Spatiotemporal Encoding (SPEN) MRI sequences.

## 🔗 Repository
You can find the source code and all related files in the official GitHub repository:
[https://github.com/andih98/Open-SPEN](https://github.com/andih98/Open-SPEN.git)

The repository contains example scripts for the following sequences:
* **SPEN-SE-EPI:** A sequence utilizing SPEN chirped-RF excitation.
* **Multi-slice Version:** An extended version with SPEN chirped-RF refocusing and optional **Diffusion-Weighting** in **Full-Refocusing**-mode.

### Educational Bloch-Simulations
To help understand the underlying SPEN-principle, the sequences are accompanied by:
* **Bloch-simulation scripts:** These are designed for deriving the **SPEN Forward Operator** for a designed sequence.
